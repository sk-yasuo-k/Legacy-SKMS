＜交通費承認画面：申請された交通費を承認する画面＞
ⅰ. 手順
以下の手順で、申請する。
1) 申請内容に問題がないならば「承認」、あるならば「申請者に差し戻す」。
2) 承認した交通費を「支払」する。
　 　
ⅱ. 状態
以下の状態が存在する。
状態 　 　 説明 　 　 　 　 　 　 　 　 　 　 　 　 　
申請済 　 　 PM/経理承認待ちの状態である。 　 　 　 　 　 　 　
PM承認済 　 経理承認待ちの状態である。。 　 　 　 　 　 　 　 　
経理承認済 　 支払待ちの状態である。 　 　 　 　 　 　 　 　 　
支払済 　 　 受領待ちの状態である。 　 　 　 　 　 　 　 　 　
受領済 　 　 申請完了の状態である。 　 　 　 　 　 　 　 　 　
※状態遷移を表示する。
ⅲ. メニュー
以下のメニューが存在する。
メニュー 　 内容 　 　 　 　 　 　 　 　 　 　 備考 　 　 　 　
承認 　 　 以下の交通費を承認する。
 PM:「申請済」の交通費
 経理:「申請済、PM承認済」の交通費		 　 　 　 　 　 　
承認前に戻す 承認前の状態に戻す。
 PM:「PM承認済」の交通費
 経理:「経理承認済」の交通費		 　 　 　 　 　 　
申請者に差し戻す 申請内容に問題があるため、申請前の状態に戻す。
 PM:「申請済」の交通費
 経理:「申請済、PM承認済」の交通費		 　 　 　 　 　 　
支払 　 　 「経理承認済」の交通費を支払う。 　 　 　 　 　 　 　 　 　 　
支払前に戻す 「支払済」の交通費を支払前の状態に戻す。 　 　 　 　 　 　 　 　
ⅳ. 検索
絞込検索をする。
項目 　 　 説明 　 　 　 　 　 　 　 　 　 　 　 　
検索 　 　 設定された検索条件で交通費を検索する。 　 　 　 　
プロジェクト （PM）検索条件に設定する、管理プロジェクトを示す。 　
　 　 　 　 （経理）検索条件に設定する、プロジェクトを示す。 　 　
全社業務 　 検索条件に設定する、全社業務を示す。 　 　 　 　 　
状態 　 　 検索条件に設定する、交通費の状態を示す。 　 　 　 　
検索条件を開く 検索条件を表示する。 　 　 　 　 　 　 　 　 　
検索条件を隠す 検索条件を表示しない。
